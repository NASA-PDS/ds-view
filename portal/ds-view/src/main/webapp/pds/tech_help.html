 Purpose

The purpose of this web site is to provide users of planetary data with the capability to search for and download any of the data associated with Mars Odyssey mission.

 Data Set Search - there are three levels of search capability:

  • MARS Data Set Quick Search - an entry level search capability which allows the user to pick one or more values from a single attribute of the data (e.g., the user can select 'Mariner69' and 'Mariner71' from the list of Missions). This interface purposely constrains the user to only selecting from a single attribute (e.g., Mission, Target, Instrument, etc).
  • MARS Data Set Advanced Search - a more advanced search capability which provides the user with more attributes against which to query for data. Also, this interface does not constrain queries to a single attribute (e.g., the user can select 'Mariner69' and 'Mariner71' from the list of Missions, can select 'Mars' as the Target, and 'Planet' as the Target Type).
  • MARS Data Set Power Search - the most advanced and powerful search capability of the three search capabilities. This interface was designed for the sophisticated user (e.g., users who are extremely familiar with the individual and collective attributes associated with the data of interest). This interface provides the widest degree of selecting attributes against which to query the data.
To process a query, select the set of attributes and click on the 'GO!' icon.

Search Results - this page displays the results of the processed query in column form:

  • Data Set - the individual names of the data sets that match the query.
  • Instrument Host - the individual names of the instrument host (spacecraft) that match the query.
  • Information About the Data Set (View Information) - by clicking on the respective link, the PDS label will be displayed for the data product.
  • Data Products and Related Files (Product Search) - by clicking on the respective link, a "default" browser will be displayed which allows the user to further constrain which data products are available for download. The "default" browser page is the entry point to where the user selects one more more files for downloading to the users local machine. The basic set of information contains:
    • Product - the list of all data products matching the query
    • Information (View Label) - clicking on the link will display the PDS label for the data product.
    • Download (Get Product) - clicking on the link will cause the web browser to display a "save file" dialog box. This is the mechanism for saving files to the user's machine.
  • Other Resources - this column displays the set of PDS resources that are available for accessing the data set. These tools provide additional information or are relevant to processing / accessing the data.

Questions and Answers:
  1. What is the PDS?

    PDS stands for the Planetary Data System. The PDS is a NASA-sponsored organization that archives and distributes high-qality planetary data to the science community.

    Back to List

  2. What is PDS-D?

    PDS-D stands for the PDS Distribution System. It is a web-based system that enables science users to find and retrieve PDS data. PDS-D is a new system, which has only been online since October 1, 2002. It is currently under active development.

    Back to List

  3. Do I need to be a scientist to use the system?

    No, but it helps. :) The system was designed for the science user, and much of the data are "low level", and require quite a bit of background knowledge to understand. If you are mainly interested in the pictures taken by various NASA missions, you are encouraged to visit the Planetary Photo Journal.

    Back to List

  4. Do I need any special software?

    Finding and downloading data through PDS-D requires only a recent version of a web browser, such as Internet Explorer or Netscape. If you plan to use the map interface to select data from the Planetary Atlas (a component of the system), you will need to install a Java 2 plug-in. For further information, see the Planetary Atlas requirements page.

    Once the data are obtained, a variety of tools may be used for viewing and analysis. Almost all PDS data are viewable with NASAView, a free program that can be downloaded from this site.

    Back to List

  5. What data are available?

    The initial version of PDS-D is focused on Mars data, and Odyssey in particular. As the system matures, it will provide access to data from other new missions and from the entire PDS archive.

    Back to List

  6. How do I find the data?

    The most common strategy is to perform a two-step search. In the first step, you narrow the search to one or more data sets of interest. In the second step, you search for products in those data sets. The place to start is the data set search page.

    Back to List

  7. How do I get the data?

    When your search returns a list of products, you can select the ones you wish to download.

    Back to List

  8. How much data can I download?

    One of the ideas behind PDS-D is that the science user can hone in on the data products of interest and not need to download an entire data set, which can be very large (THEMIS is expected to grow to around 6 terabytes. If your download requirements exceed 500 megabytes, it is suggested that you contact the PDS_Operator.
