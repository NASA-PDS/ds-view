Transport Service (Upload)

The Transport Service provides functionality for uploading files and writing them to a staging area.

  • Service Interface

    The service only supports "multipart/form-data" HTTP POST requests which must include a "file" part containing the name of the file to be uploaded. Additional information and examples can be found in the service's Operation document.
