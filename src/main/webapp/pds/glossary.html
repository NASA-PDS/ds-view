PDS Glossary
 
Archive Status: provides the status of a data set that has been submitted for inclusion into the PDS archive.

Curator: provides the name of the node or responsible organization currently maintaining the data set or volume and is responsible for maintaining catalog information.

Data Engineer: provides the name of the Central Node data engineer.

Data Object Type: identifies the data object type of a given set of data. Example values: image, map, and spectrum.

Data Set ID: is a unique alphanumeric identifier for a data set or a data product. The data_set_id value for a given data set or product is constructed according to flight project naming conventions. In most cases the data_set_id is an abbreviation of the data_set_name.

Data Set Name: provides the full name given to a data set or a data product. The data_set_name typically identifies the instrument that acquired the data, the target of that instrument, and the processing level of the data.

Instrument Host Name: provides the full name of the host on which an instrument is based. This host can be either a spacecraft or an earth base. Thus, the instrument_host_name element can contain values which are either spacecraft_name values or earth_base_name values.

Instrument Host Type: provides the type of host on which an instrument is based. For example, if the instrument is located on a spacecraft, the instrument_host_type element would have the value SPACECRAFT.

Instrument Name: provides the full name of an instrument. Note: that the associated instrument_id element provides an abbreviated name or acronym for the instrument.

Instrument Type: identifies the type of an instrument.

Mission Name: identifies a major planetary mission or project. A given planetary mission may be associated with one or more spacecraft.

Start Date: provides the date of the beginning of an event or observation (whether it be a spacecraft, ground-based, or system event) in UTC system format. Formation rule: YYYY-MM-DD.

Stop Date: provides the date of the end of an observation or event (whether it be a spacecraft, ground-based, or system event) in UTC system format. Formation rule: YYYY-MM-DD.

Target Name: identifies a target. The target may be a planet, satellite, ring, region, feature, asteroid, or comet.

Target Type: identifies the type of a named target. Example values: planet, satellite, ring, region, feature, asteroid, and comet.
