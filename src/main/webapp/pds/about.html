Introduction

PDS-D is a new Web application that provides seamless access to archive data from heterogeneous, geographically disperse data repositories and systems. Data from the Mars Odyssey mission and all subsequent missions will be made available through the PDS-D immediately upon delivery to the PDS.

System Capabilities

  • Unified web-based user interface for novice and experienced users
  • Convenient search-and-retrieval functions
  • Access to a large variety of instrument and data types
  • Data distribution through the Internet
  • Subscription/notification services to enable users to track new data releases
  • Technical help via HELP, FAQ, About pages and email
